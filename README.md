**HeroFrame AI** — agentic product ad studio (Next.js app).

This repo is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Environment

Copy `.env.example` to `.env.local` and add your keys. **Do not commit** `.env`, `.env.local`, or any file containing secrets.

## Security

See [SECURITY.md](./SECURITY.md). API routes are **unauthenticated** by design for this demo; lock them down before any public deployment that could burn paid quotas.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
